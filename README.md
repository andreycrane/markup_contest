# markup-contest

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Deploy project to firebase hosting
```
npm run deploy
```

### Install package for converting images to webp format
```
sudo apt-get install webp
```
